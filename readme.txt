Add a nice graph with your published posts during the last 30 days on a dashboard widget.

== Description ==

Adds a simple but useful chart with the published posts during the last 30 days to your admin dashboard.

== Installation ==

1. Upload extracted folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done! You can check your stats on the Dashboard
